# JNNL
Java Neural Network Library
A simple Neural Network library I'm working on for fun. 

<b>Features:</b>

  +Sigmoid, Tanh and Rectifier (ReLU) activation functions.

  +Weighted summing function.

  +Bias neuron on all hidden layers.

  +Backpropagation.
